URL of a document

27 November 2010

Introduction

This topic defines how a URL of a document can be modeled.

Model

An instance of FunctionalPhysicalObject FPO-123509 is represented by the instance of ClassOfInformationRepresentation COIR-439822, that is classified as a P&ID (Piping and Instrument Diagram) and is identified as HL-350454 rev. 2.

In order to find its record on some server, being an instance of MaterializedPhysicalObject that is a member of COIR-439822, we represent that record with the URL http://www.XYZ-ltd.com/documents/HL-350453-rev.2.jpg  .

A document record on a disk can be accessed through a URL

NOTE - It is very unlikely that you would be interested in that identifier of the record on the disk. It will only be of any interest in case you express this information with a longhand ('lifted') template. In that case let the software create some phony, but unique!, ID.
